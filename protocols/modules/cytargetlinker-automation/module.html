Cytargetlinker Automation

Extend a network in an automated manner

Introduction

  • CyTargetLinker provides a quick and extensive enrichment of biological networks in Cytoscape
  • The visualization options allow biological interpretation of complex regulatory networks in a graphical way
  • In this tutorial a step-by-step explanation is given on how to extend a list of microRNAs with predicted and validated microRNA-target interactions (MTIs)

Step 1a: Import a set of microRNAs into Cytoscape

  • First you will need to create a biological network in Cytoscape 3.0 and higher containing your microRNAs of interest.
  • The datafile should contain at least two columns, (i) the microRNA name and (ii) the corresponding miRBase accession number.
  • An example datafile, in which two microRNAs known to be involved in obesity are listed, can be found here
  • Open the datafile containing the microRNAs in Cytoscape via File -> Import -> Network ->File.
  • First you select the file that you want to load into Cytoscape. Thereafter, a panel becomes visible were you should select the microRNA names as Source Nodes and miRBase acc-number as Source Node Attribute, as shown on your right.
